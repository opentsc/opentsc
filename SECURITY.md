# 安全与隐私 / Security & Privacy

## 数据归属

OpenTSC 是**单人、本地优先（local / offline-first）**系统。框架本身不收集、不上传任何数据；你创建的 vault（`soul/`、`world/` 等）默认只存在于你自己的机器上。`.gitignore` 已排除常见 vault 数据目录，避免误提交真实数据。

## 本仓库不含真实个人数据

这是**框架仓库**。所有示例（Carol / Dave / Eve / Acme 等）均为占位名，不对应真实个人。**请勿向本仓库提交任何真实姓名、聊天记录、联系方式或可识别个人的情报。**

## 负责任使用

- **不要**用它对未经同意的个人进行画像、监控或起底。
- **不要**抓取第三方私域数据（他人聊天记录）评估真实个人。
- 遵守你所在司法辖区的隐私法（GDPR、个人信息保护法 PIPL 等）。

## 报告漏洞

发现安全/隐私问题，请通过 GitHub Security Advisory 私下报告，不要公开 issue。
